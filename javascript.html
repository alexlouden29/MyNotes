Miscellaneous

  • // For single line comment, /* for multiline comment
  • Variable name convention leaves the first word lowercase, rest upper case
  • keyword "var" to make any kind of variable
  • Semicolons are not need to end a line, but can be used, and maybe should be
  • Use \ to escape a character, example \'
  • === to compare equality, and !== for not equals. Why it is three instead of two, I still wonder
  • Use && || for joining conditions in if statements (and, or)
  • Example of switch statement:
    switch(condition) {
    case 'case1': doSomething();
    default: doSomethingElse();
    }
  • Example of function declaration:
    function myfunc (myParam1, myParam2){
    return myParam1 + myParam2
    }

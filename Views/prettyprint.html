Pretty Print

Include script line into head for access to pretty print, then place code withing pre element as follows: 

		<script src="https://cdn.rawgit.com/google/code-prettify/master/loader/run_prettify.js"></script>
		<pre class="prettyprint">
			x = 2;
		</pre>

To change color scheme, at the end of the src URL add "?skin=sunburst" or another skin. Options include sunburst, desert, sons-of-obsidian, doxy

To remove border, include CSS as shown: 

		pre.prettyprint {
    			border: none; 
		}

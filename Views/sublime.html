SUBLIME NOTES

Shortcuts

Shortcut function
Ctrl + l Select current line
Alt + . Close current tag
Ctrl + u Return to last selection
Ctrl + Shift + up/down Swap line positions
Ctrl + Shift + l (With highlighted Selection) Break selection into multiple cursors
Ctrl + left/right Navigate by word
Ctrl + PageUp/Down Navigate by tab

Other Notes

  • Type "html" then press tab to get an automatically generated boilerplate.
    • Make sure the file type in the bottom right corner is set to "HTML" The boiler plate does not include ""

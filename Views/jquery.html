The Basics 

     
		$(document).ready(function(){
    			jQuery magic;
		});
    • This is the basic starter. $() is the jquery starter, and putting "document" In there tells it you will be interacting with the DOM.
    • .ready() is one of several possible functions, in this case it says to do the contents of the () when the document is ready.
    • So, function will happen when the document is ready (loads) (observe syntax of function declaration inside the ready function)
  • Example jQuery magic: $('div').slideDown('slow');
    • $() is used to do anything with jQuery
    • for any selector, use '' a la 'div'

Functions

  • .ready(stuff)
    • Executes stuff when document is loaded. Stuff can included a function declaration
  • .slideDown('slow')
    • Has element slide downward

Hyper Text Markup Language

Basics

  • Comment: <!-- This is a comment -->

Starting a new file: BoilerPlate Code

  • Start with "<!DOCTYPE html>"
  • Include "<html>"" tag to ensure correct interpretation
  • and of course "<body>""
  • example: 
     	
<!DOCTYPE html>
<html lang="en">
 	<head>
   	<title> My Title</title>
 	</head>
 	<body>
 	</body>
</html>
  • Escape Characters
    • " becomes & quot;
    • & becomes & amp;
    • < becomes & lt;
    • > becomes & gt;
    • All encodings

    Tags

    • <br> for a page break
    • <blockqoute> for a weird spaced indented piece of text
    • <a> has attribute "target". target="_blank" opens link in a new page.
      Wrap any element (text or image for example) with this tag to make it a clickable link.
    • <img src=''/> Can be used for XSS (Cross-site Scripting)
      "alt" tag can be used to create a description of the image, for visually impared, or if image does not load
    • <style> Can be used to put CSS elements directly into an HTML file
    • <link> Used to link an HTML file to another source, such as a CSS file. Example: <link href="/style.css" type="text/css" rel="stylesheet"> Link elements do not have content like the a element.
      • If the css file is out one folder and then in another, use ../ (like linux shell) to go back one folder.
    • <pre> Keeps the formatting as it is typed in the html file. Used for stuff with weird formatting (and prettyprint).
    • <li>
    • <ol> and <ul>
